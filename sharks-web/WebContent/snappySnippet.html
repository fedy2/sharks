Database on Port State Measures

Measures from 51 Port States are available.


The Database on Port State Measures (Port-Lex) provides access to port state measures (PSM) adopted by states to prevent, deter and eliminate Illegal, Unreported and Unregulated fishing (IUU fishing). Information can be accessed through a simple word search or advanced search using country names or measure components. See the coverage section for more on the search terms. [more...]
